Private Polls

Private polls is a premium feature that we plan to roll out soon.

Private polls allows users to create a poll and select the people that can vote on it, with the poll remaining visible only to those selected.

It will also allow users to create groups so that only members of that group can vote. For example, this could be used by a workplace to gauge opinion on who should be their union representative, or by local residents to vote on whether a development should go ahead.
